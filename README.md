# Scanner-go & Subfinder CLI Checker 🚀

A simple yet powerful CLI tool written in **Python** to automate subdomain discovery and vulnerability scanning. 

## 🛠 Installation & Usage

Copy and paste the commands below to get started:

```bash
# Clone the repository (if not already)
git clone [https://github.com/kuhujiban1c/scanner)
cd scanner

# Install dependencies
pip install -r requirements.txt

# Set permissions and run the installer
chmod +x install.sh
./install.sh

# Run the tool
python3 main.py
